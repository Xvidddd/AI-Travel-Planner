# UI 与品牌设计指南

## 1. 品牌定位
- **名称**：AuroraVoyage
- **调性**：明亮、轻盈、科技自然融合，灵感来自极光与航线。
- **差异点**：example 项目偏暗色 + shadcn/ui 设计；本项目采用浅色渐变 + 玻璃拟态卡片 + 地图为核心，减少阴影与复杂边框。

## 2. 色彩体系
| 用途 | 颜色 | 说明 |
| --- | --- | --- |
| 主色 | `#3B82F6` (Aurora Blue) | 关键按钮、活跃标签 |
| 强调色 | `#F97316` (Sunset Orange) | 预算预警、语音按钮脉冲 |
| 背景渐变 | `#F8FBFF → #D6EEFF` | 全局背景，制造云朵感 |
| 地图浮层 | `rgba(255,255,255,0.85)` | 地图上方信息卡 |
| 中性色 | `#0F172A`, `#475569`, `#CBD5F5` | 文本与边界 |

## 3. 字体与排版
- **中文**：思源黑体（或苹方/鸿蒙体 fallback）。
- **英文**：Inter。
- **字号层级**：H1 30px、H2 24px、Body 16px、Caption 13px。
- **行程时间线**：使用等宽字体段落展示时间，搭配彩色节点。

## 4. 布局规范
- **整体布局**：
  - 左侧 60% 宽度为地图画布（固定），右侧 40% 采用多层 Tab（行程/费用/记账/设置）。
  - 顶部导航为透明玻璃条（模糊背景 18px），包含 Logo、行程切换、用户菜单。
  - 底部浮动语音面板：包含麦克风按钮、语音波纹、文本确认区。
- **响应式**：
  - ≥1280px：三列布局（地图 + 时间线 + 费用卡）。
  - 768-1279px：地图占上半屏，卡片在下方滚动。
  - <768px：Tab 变为底部导航，语音按钮悬浮右下角。

## 5. 组件策略
- **Button**：圆角 999px，背景半透明，hover 叠加内部阴影。
- **Card**：有 12px 圆角、1px 内描边、背景透明度 0.85，支持内容分组。
- **Timeline**：彩色节点 + 竖向线条 + 卡片联动；点击节点在地图上高亮。
- **Voice Control**：环形按钮 + 动态脉冲（CSS animation），录音时展示实时字幕。
- **地图 POI**：使用柔和色块标记，不同类别（景点/美食/酒店/交通）对应不同图标。

## 6. 动效与微交互
- 语音录制：rAF 驱动的渐变声波，结束录制后平滑收缩。
- 行程卡片：加载时 120ms 淡入与上移 8px。
- 地图标记：新增/高亮时放大 1.2x 后回落。
- 预算进度条：超预算触发颜色从蓝色渐变至橙色并轻微震动 2 次。

## 7. 设计交付物
- 低保真线框：Figma (或 Excalidraw) 展示三种布局。
- 高保真组件库：包含按钮、卡片、时间线、语音面板、地图气泡。
- 交互动效：Lottie / JSON 描述或 CSS 规范。

本指南将指导后续前端实现，确保在保持功能一致的同时呈现全新视觉体验。
